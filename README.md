# Cibersecurity-Tool-Guide

## Scanning tools

### nmap

It is a widely used network scanning and auditing tool in cybersecurity and systems administration.

### arp-scan

It is a tool designed to discover devices on a local area network (LAN) using the Address Resolution Protocol (ARP).

### masscan

It is an extremely fast port scanner designed to scan large numbers of hosts in a short amount of time.
